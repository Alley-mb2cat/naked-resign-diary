# 🌿 裸辭日記

> 透過目標、規劃與每日反思，降低裸辭路上的焦慮與不安。

## 功能

- **裸辭評估測驗** — 資金面、心理面、目標面三維度評估
- **目標設定** — 最多 5 個目標，含快速模板
- **靈感庫** — 三種任務類型：單次任務、成長軌跡、體驗
- **每日日記** — 滑動卡片：目標進度 → 心情紀錄 → 感恩日記（支援多目標選取）
- **微光集** — 收藏每天的美好時刻（支援照片）
- **前進軌跡** — 月曆視圖 + 統計數據
- **給自己的信** — 寫給未來低潮時的自己
- **肯定語系統** — 完成日記後隨機鼓勵

## 部署

### Netlify（推薦）
1. Fork 或 clone 這個 repo
2. 到 [Netlify](https://app.netlify.com) → New site → Import from Git
3. 選擇這個 repo，Publish directory 設為 `.`
4. 部署完成，拿到網址

### 更新
```bash
# 改完檔案後
git add .
git commit -m "update"
git push
# Netlify 自動部署
```

## 加到手機主畫面

### iOS
Safari 打開 → 分享 → 加入主畫面

### Android
Chrome 打開 → 選單 → 安裝應用程式

## 資料儲存
所有資料存在瀏覽器 localStorage，關閉不會消失。
